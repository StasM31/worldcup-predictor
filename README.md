# ⚽ ЧМ Прогнозы — Инструкция по запуску

## Что это
Сайт для турнира прогнозов на матчи чемпионата мира. Каждый участник ставит счёт перед матчем, после того как все поставили — Telegram-бот рассылает прогнозы всем участникам.

**Очки:**
- 🎯 Угадал точный счёт → **3 очка**
- ✅ Угадал исход (победитель или ничья) → **1 очко**

---

## Шаг 1 — Создать Telegram-бота

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`, придумай имя и username (например `wc_predictions_bot`)
3. Скопируй **токен бота** (выглядит как `1234567890:ABCdef...`)

---

## Шаг 2 — Деплой на Railway (бесплатно)

1. Зайди на [railway.app](https://railway.app) и зарегистрируйся (через GitHub)
2. Нажми **"New Project" → "Deploy from GitHub repo"**
3. Загрузи файлы этой папки в новый GitHub репозиторий (или используй Railway CLI)
4. В Railway перейди в настройки проекта → **Variables** и добавь:

| Переменная | Значение |
|---|---|
| `TELEGRAM_BOT_TOKEN` | токен от @BotFather |
| `ADMIN_TOKEN` | придумай сложный пароль (это твой пароль администратора) |
| `PORT` | `8000` |

5. Railway автоматически запустит сайт. Скопируй URL (вида `https://xxx.railway.app`)

---

## Шаг 3 — Подключить Telegram webhook

После деплоя, один раз выполни в терминале:

```bash
TELEGRAM_BOT_TOKEN="твой_токен" APP_URL="https://xxx.railway.app" python setup_webhook.py
```

Или просто открой в браузере:
```
https://api.telegram.org/botТВОЙ_ТОКЕН/setWebhook?url=https://ВАШ_САЙТ/api/telegram/webhook
```

---

## Шаг 4 — Первый вход (администратор)

1. Открой сайт
2. В поле токена введи свой **ADMIN_TOKEN** (пароль из шага 2)
3. Появится вкладка ⚙ **Админ**

---

## Шаг 5 — Добавить участников

1. В админке → раздел **Участники** → введи имя → нажми **Добавить**
2. Появится кнопка **📋 Ссылка** — скопируй и отправь участнику
3. По этой ссылке участник сразу войдёт на сайт

---

## Шаг 6 — Подключить Telegram участников

Каждый участник должен:
1. Открыть твоего бота в Telegram (например `t.me/wc_predictions_bot`)
2. Отправить команду: `/start ТОКЕН_УЧАСТНИКА`
   (токен виден в таблице участников в админке)

После этого бот подтвердит подключение, и участник будет получать уведомления.

---

## Как пользоваться

### Администратор:
- **Добавить матч**: введи команды и время → нажми Добавить
- **Начало матча**: нажми **▶ Старт** (участники больше не смогут ставить)
- **Ввести результат**: нажми **Ввести счёт** → введи счёт (например `2:1`) → очки начисляются автоматически

### Участники:
- Открывают сайт по своей ссылке
- Ставят прогноз на каждый матч (пока матч "Скоро")
- Как только все поставили — бот рассылает прогнозы всем в Telegram
- После матча видят результаты и очки в рейтинге

## Локальный запуск (для тестирования)

```bash
pip install -r requirements.txt
ADMIN_TOKEN=admin123 TELEGRAM_BOT_TOKEN=xxx uvicorn main:app --reload
# Открой http://localhost:8000
```
