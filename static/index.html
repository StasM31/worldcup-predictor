🏅 Определение победителя

Побеждает участник, набравший наибольшее количество очков по итогам турнира.

При равенстве очков преимущество получает участник:

  • с большим количеством точно угаданных счётов 🎯
  • при дальнейшем равенстве — с большим количеством угаданных исходов ✅
  • при дальнейшем равенстве — призовые места делятся поровну

  • Точный счёт → 9 очков

🌍 Бонусы за весь турнир

  • Угадал чемпиона → +10 очков
  • Угадал финалиста (за каждого) → +5 очков
  • Угадал команду на 3-м месте → +5 очков

Прогноз на турнир принимается до начала первого матча.

⏱ Основное время

Прогноз делается только на счёт основного времени матча (90 минут).

Например: основное время завершилось со счётом 1:1, в дополнительное время забили ещё гол — 2:1. Расчёт ставки ведётся по счёту основного времени, то есть 1:1.

⏰ Дедлайн ставок

Прогнозы принимаются в течение 1 минуты после официального старта матча. После — закрыто автоматически.

Не успел → 😴 0 очков за матч.

📖 Как сделать прогноз

1
Открой вкладку «Матчи»
Найди матч со статусом ● Скоро — это значит ставки принимаются.
2
Введи счёт
Нажми на поле счёта и введи цифры — сначала голы хозяев, затем гостей.
3
Нажми кнопку «Поставить»
После нажатия поля счёта станут зелёными — это значит прогноз принят. До начала матча можно изменить прогноз кнопкой «Изменить».
!
Не забудь про ва-банк
Один раз за весь турнир можно нажать 🔥 Ва-банк вместо «Поставить» — очки утраиваются!
🏆
Не забудь про долгосрочный прогноз!
Зайди на вкладку «Ставка на финал» и угадай чемпиона, финалистов и команду на 3-м месте. Прогноз принимается только до начала первого матча — потом закрывается навсегда!

Чтобы настроить бэкап, сделай один раз:
1. Напиши боту @userinfobot в Telegram — он ответит твоим числовым ID (например: 123456789)
2. Зайди на railway.app → сервис web → вкладка Variables → New Variable
3. Name: BACKUP_CHAT_ID, Value: твой числовой ID из шага 1
4. Нажми Add — готово!
